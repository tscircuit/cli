# @tsci/dopamide.basic2

A tscircuit component package.

## Getting Started

### Prerequisites

- [Bun](https://bun.sh/) or Node.js 20+
- TSCircuit CLI (`bun install -g @tscircuit/cli`)

### Installation

```bash
bun install
```

### Development

Start the development server:

```bash
bun run dev
```

This will open an interactive preview where you can view your circuit's schematic and PCB layout.

### Building

Build the circuit:

```bash
bun run build
```

### Testing

Run snapshot tests:

```bash
bun run snapshot
```

Update snapshots:

```bash
bun run snapshot:update
```

## Publishing

Push to the tscircuit registry:

```bash
tsci push
```
